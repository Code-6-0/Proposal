\subsection{User Research}

Untuk memahami kebutuhan pengguna, telah dilakukan penelitian kualitatif dan kuantitatif sebagai berikut:

\begin{enumerate}[leftmargin=*]
    \item \textbf{Wawancara} — Dilakukan wawancara mendalam dengan 5 UMKM kuliner di Kota Bandung untuk memahami tantangan dalam pengelolaan makanan surplus.
    
    \item \textbf{Survei Online} — Disebarkan kuesioner kepada 100 responden (konsumen potensial) untuk mengetahui preferensi pembelian makanan surplus.
    
    \item \textbf{Observasi} — Pengamatan langsung terhadap operasional UMKM kuliner selama 3 hari untuk mengidentifikasi pain points.
\end{enumerate}

\subsubsection{Temuan Utama}
\begin{itemize}[leftmargin=*]
    \item 78\% UMKM kesulitan memprediksi jumlah produksi harian
    \item 65\% UMKM membuang 10-20\% produk harian karena tidak terjual
    \item 82\% konsumen bersedia membeli makanan surplus dengan harga diskon 30-50\%
    \item 90\% konsumen membutuhkan jaminan kelayakan makanan sebelum membeli
\end{itemize}

\subsection{Prinsip Desain yang Diterapkan}

\begin{enumerate}[leftmargin=*]
    \item \textbf{Konsistensi} — Penggunaan warna, tipografi, dan komponen UI yang konsisten di seluruh halaman.
    \item \textbf{Hierarki Visual} — Informasi penting seperti harga diskon dan sisa waktu ditampilkan dengan ukuran dan warna yang menonjol.
    \item \textbf{Feedback} — Setiap aksi pengguna mendapatkan konfirmasi visual (toast notification, loading indicator).
    \item \textbf{Affordance} — Tombol dan elemen interaktif dirancang intuitif sehingga pengguna tahu apa yang dapat diklik atau diinteraksi.
    \item \textbf{Error Prevention} — Formulir dilengkapi validasi real-time untuk mencegah input yang salah.
\end{enumerate}

\subsection{Fitur Accessibility}

\begin{itemize}[leftmargin=*]
    \item \textbf{Color Contrast} — Mengikuti standar WCAG 2.1 dengan rasio kontras minimal 4.5:1 untuk teks normal.
    \item \textbf{Keyboard Navigation} — Seluruh fitur utama dapat diakses menggunakan keyboard.
    \item \textbf{Alt Text} — Semua gambar memiliki deskripsi alternatif untuk screen reader.
    \item \textbf{Responsive Design} — Tampilan optimal di berbagai ukuran layar (mobile, tablet, desktop).
    \item \textbf{Font Size} — Ukuran font minimal 14px untuk keterbacaan di mobile.
\end{itemize}
