\subsection{Scope yang Tidak Dicakup}

Fitur-fitur berikut sengaja di-\textit{exclude} dari versi kompetisi ini dan direncanakan untuk pengembangan selanjutnya:

\begin{enumerate}[leftmargin=*]
    \item \textbf{Sistem Prediksi Produksi Harian} — Membutuhkan data historis yang cukup dan validasi model yang lebih panjang.
    \item \textbf{Mekanisme Donasi Otomatis} — Memerlukan kerjasama dengan lembaga donasi dan verifikasi penerima.
    \item \textbf{Fitur Chat Real-time} — Membutuhkan infrastruktur tambahan dan moderasi konten.
    \item \textbf{Mobile Application} — Fokus pengembangan saat ini adalah versi web responsive.
    \item \textbf{Integrasi Payment Gateway} — Menggunakan simulasi pembayaran untuk versi kompetisi.
\end{enumerate}

\subsection{Keterbatasan Teknis}

\begin{enumerate}[leftmargin=*]
    \item \textbf{Platform} — Aplikasi dikembangkan sebagai web application yang dapat diakses melalui browser di desktop dan mobile (responsive design).
    
    \item \textbf{Koneksi Internet} — Aplikasi membutuhkan koneksi internet yang stabil untuk berfungsi optimal. Mode offline belum didukung.
    
    \item \textbf{Browser Support} — Dukungan penuh untuk Chrome, Firefox, Safari, dan Edge versi terbaru.
    
    \item \textbf{Skalabilitas Awal} — Infrastruktur saat ini dirancang untuk mendukup hingga 1.000 pengguna simultan. Optimasi untuk skala lebih besar akan dilakukan setelah kompetisi.
    
    \item \textbf{Data Historis} — Sistem membutuhkan minimal 7 hari data transaksi untuk menghasilkan statistik dan laporan yang akurat di dashboard UMKM.
\end{enumerate}
